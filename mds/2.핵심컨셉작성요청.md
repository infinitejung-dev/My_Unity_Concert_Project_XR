# 요구사항

1. `기획기본구조.md`의 '핵심 컨셉'을 작성해줘.

## 컨셉 개요
- 특정 공간을 하나의 좌표계로 구성한다.
- 공간에 존재한 각각의 사람에게 좌표를 부여한다.
- 각 사람의 눈 위치를 기준으로 동일한 3D 오브젝트를 서로 다른 시점과 각도로 렌더링한다.
- 공간 중앙의 마커를 모두가 바라보게하여 3D 오브젝트가 실제 공간에 위치하는 것처럼 느끼게 한다.
- 모든 사람의 화면이 동기화되어 이벤트가 일어나면 각각의 디바이스가 같은 3D 오브젝트를 렌더링한다.

개발자가 알아보기 쉽게 최대한 문장을 장황하게 길게 쓰지 말고, 구조화해서 작성해줘.
