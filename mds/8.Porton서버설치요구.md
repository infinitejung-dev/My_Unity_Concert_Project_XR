# 역할
너는 10년차 유니티 서버 개발자야.
다수의 Photon Fusion 서버를 유니티에 만들었어.
AI를 활용한 개발에 능숙해.

## 배경
`Assets/mds/단계별수행절차.md`에 따라 프로토타입을 제작했어.
Photon Fusion 서버를 설치해서 이 프로토타입의 'StageObject_Halo'의 위치를 동기화 시키려고해.

## 요구사항

- 아래의 기능들을 구현할 수 있는 서버를 만드는 방법을 각 task 단위로 분리해서 `Assets/SVTasks` 폴더에 MD 파일로 만들어줘.
- 각 task의 구현을 제대로 검증하기위해 체크리스트를 만들어서 `Assets/SVChecklists` 폴더에 md 파일로 만들어줘.
- 한 번에 하나의 task를 구현하고, 유니티 에디터에서 테스트하는 방식으로 진행할거야.
  - 한 번에 하나의 task가 끝나면 다른 task를 구현한다.
- task에는 구현 순서 번호를 붙여줘.


### photon fusion 서버 기능
- 클라이언트 3명이 동시에 접속할 수 있다.
- 서버에서 동기화 시킬 대상은 1개의 게임오브젝트이다.
	- 위치, 회전, 크기 값을 동기화 속성으로 한다.
- 개인 PC를 포톤 서버로 사용한다.
  
### 클라이언트 기능
- 스마트폰에서 카메라를 사용해 화면을 비추면 월드에 있는 물체를 화면에 띄워준다.
