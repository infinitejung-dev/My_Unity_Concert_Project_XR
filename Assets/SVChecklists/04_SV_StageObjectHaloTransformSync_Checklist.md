# Checklist 04. `StageObject_Halo` 위치/회전/크기 동기화 검증

## 구현 확인
- [ ] 위치 동기화가 구현되어 있다.
- [ ] 회전 동기화가 구현되어 있다.
- [ ] 크기 동기화가 구현되어 있다.
- [ ] Host만 값 변경 권한을 가진다.
- [ ] 테스트용 이동/회전/스케일 입력 또는 디버그 컨트롤이 있다.

## 동작 검증
- [ ] Host에서 오브젝트를 이동하면 모든 클라이언트에서 같은 월드 위치로 반영된다.
- [ ] Host에서 오브젝트를 회전하면 모든 클라이언트에서 같은 월드 회전으로 반영된다.
- [ ] Host에서 오브젝트 크기를 바꾸면 모든 클라이언트에서 같은 크기로 반영된다.
- [ ] 늦게 접속한 클라이언트도 최신 위치/회전/크기를 받는다.
- [ ] 접속 중 짧은 네트워크 지연이 있어도 상태가 최종적으로 수렴한다.

## 프로토타입 맥락 검증
- [ ] 동기화 값은 관객 로컬 기준이 아니라 공연장 월드 기준이다.
- [ ] 오브젝트 권위 값 1개를 관객별 카메라가 다르게 보는 구조를 유지한다.

## 완료 판정
- [ ] `StageObject_Halo` 단일 오브젝트 동기화 요구가 충족됐다.
